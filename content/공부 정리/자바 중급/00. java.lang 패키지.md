### `java.lang`
- 자바가 기본으로 제공하는 라이브러리 중에 가장 기본이 되는 것
- `lang` 은 language 의 줄임말
- 가장 기본이 되는 패키지로써 `import` 가 생략 된다.

### `java.lang` 패키지의 대표적인 클래스
- `Object` : 모든 자바 객체의 부모 클래스
- `String` : 문자열
- `Integer`, `Long`등등 : 래퍼 타입, 기본형 데이터 타입을 객체로 만든 것
- `Class` : 클래스 메타 정보
- `System` : 시스템과 관련된 기본 기능들을 제공
